# 1 用户登录/注册
## 1.1 [POST] /api/user/login
### 入参
```JSON
{
    "username": "string",   // 用户名，用户用于登录的账户名
    "password": "string"    // 密码，用户的账户密码
}
```
### 出参
```JSON
{
  "code": 200,      // 响应状态码，200表示请求成功
  "message": "Login successful",  // 响应信息，返回登录成功或失败的提示
  "token": "string"     // 鉴权token，登录成功后生成，用于后续接口的鉴权
}
```
